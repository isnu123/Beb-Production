# Portofolio Beb Production - Creative Video & Streaming

Proyek ini adalah Website Portofolio Bisnis resmi untuk **Beb Production**, sebuah entitas kreatif yang didirikan pada tahun 2024 yang berfokus pada layanan live streaming dan produksi video profesional. Proyek ini disusun khusus untuk memenuhi tugas **Ujian Tengah Semester (UTS)** mata kuliah Pemrograman Web.

## 🚀 Fitur Utama Website
* **Home:** Halaman perkenalan dengan narasi filosofi bisnis dan keunggulan kompetitif Beb Production.
* **About Us:** Penjelasan mendalam mengenai visi, misi, dan sejarah berdirinya perusahaan di Wonogiri.
* **Services:** Daftar layanan teknis lengkap, mulai dari Live Streaming, Video Production, hingga Multi-Camera Setup.
* **Contact Us:** Integrasi saluran komunikasi langsung melalui WhatsApp, Instagram, dan YouTube.
* **Single Page Scroll Experience:** Fitur perpindahan antar seksi yang halus (smooth scrolling) untuk kenyamanan pengguna.
* **Modern UI/UX:** Menggunakan tema Dark Mode yang elegan dengan efek *Glassmorphism* dan animasi branding yang dinamis.

## 🛠️ Teknologi yang Digunakan
* **HTML5 & CSS3:** Struktur utama dan styling kustom (Variabel CSS, Flexbox, & Grid).
* **Bootstrap 5.3:** Framework CSS untuk memastikan website bersifat responsif di semua ukuran layar.
* **JavaScript:** Digunakan untuk logika navigasi, fitur scroll snap, dan animasi Navbar.
* **Google Fonts (Poppins):** Tipografi standar profesional agar teks mudah dibaca.
* **Bootstrap Icons:** Ikonografi visual untuk memperjelas setiap poin layanan.
